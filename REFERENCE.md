# Luna Llena — External Reference (award-grade, last ~18 months)

Per the Blocklist Law, the bar anchors ONLY to external award-grade reference, never
past builds. Studied for the moonlit-coastal beauty-house direction:

1. **Editorial salon lookbooks (Awwwards/SiteInspire hair-salon nominees)** — large
   full-bleed work photography, minimal nav, sections named for the craft not "About
   / Services." Takeaway: let the WORK be the color; type stays quiet and large.
2. **Coastal / lunar wellness & spa sites (Godly, current)** — soft radial light
   sources, deep-night-to-moonlight gradients, generous negative space, slow ambient
   motion. Takeaway: a single luminous light source reads as calm + expensive.
3. **Masonry editorial galleries (SiteInspire fashion/beauty)** — staggered-rise
   masonry walls with mixed aspect tiles. Takeaway: masonry must have rhythm (varied
   spans), not a uniform grid, to read editorial rather than ecommerce.
4. **Bilingual neighborhood-business sites** — small ES/EN kicker pairs that signal
   "this is for you" to the real block without translating the whole page.

## What this build takes (the bar, not the shapes)
- A SINGLE luminous light source (the full moon) as the hero's organizing idea.
- Masonry-with-rhythm gallery of real salon work as the medium.
- Quiet, large Supreme type; the work carries the color.
- Coastal-clean palette: moonlight white, sand, deep teal-night, teal #14868C edge.
- Bilingual ES/EN kickers, never a full translation.

## Cliché bans honored (anti_rules / nail-salon+spa lens)
NO pink florals, NO lotus clip art, NO Edwardian script, NO pastel gradient blobs,
NO discount-shop look, NO generic stock carousel. The moonlit-coastal direction is
the deliberate anti-default.

## Signature (industry-rare)
Breathing full-moon light source raking moonlight across a masonry wall of real
work on scroll. Not seen on a Main St unisex salon site.
